Python Overview
Python is a programming language named after Monty Python; not the snake. Beyond basic syntax, there are two notable features that need to be understood for this assignment:

[List] Comprehensions (Part 1): Allows transforming iterable values using map/filter logic, e.g. [num * num for num in nums if num % 2 == 0].
Decorators (Part 2): Allows wrapping a function to customize it's behavior, e.g. @cache def fib(n): ....
Note that when decorators have arguments, the precedence is @(returnsDecorator("argument")) def function(): ..., i.e. the "decorator" is really a function call that returns the real decorator used by @.
